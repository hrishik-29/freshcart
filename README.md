# FreshCart

## Overview
FreshCart is an interactive, frontend-focused grocery application built with modern web technologies. This project was developed with a strong emphasis on Human-Computer Interaction (HCI) methodologies to deliver an exceptional, user-friendly shopping experience. 

## Key HCI Enhancements
- **Skeuomorphism**: Subtle design cues that mimic real-world interactions.
- **F-Pattern Layout**: Optimized visual hierarchy for natural eye-scanning.
- **Goal-Gradient Effect**: Visual progress indicators for checkout flows.
- **Peak-End Rule**: Delightful micro-animations and seamless error handling to ensure user satisfaction.

## Getting Started

### Prerequisites
- Node.js (v16+)
- npm

### Installation
1. Clone the repository and navigate to the folder.
2. Install the necessary dependencies:
   ```bash
   npm install
   ```

### Running the App
To start the local development server:
```bash
npm run dev
```
Navigate to `http://localhost:5173` (or the port specified in your terminal) to view the application.
